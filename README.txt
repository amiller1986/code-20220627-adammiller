bmi.py contains a function to convert a json string of patient data into a pandas dataframe and then to calculate BMI and risk.
A basic unit test class is included.

Prototyping was done in a jupyter notebook. The ipynb is attached for reference.

If using anaconda package manager, you can import the environment by using:

conda env create -f bmi_env.yaml

at the anaconda prompt.

Run using python3 environment setup according to bmi_env.yaml. 

e.g. on the command line:

python bmi.py
